Standard Operating Procedures

Detailed step-by-step procedures for VMware administration, Windows Server management, and Azure hybrid cloud tasks — built from real enterprise experience.

ESXi Host Deployment

VMware · Infrastructure
  1. Boot from ISO: Mount ESXi ISO via iDRAC/iLO virtual media, boot server
  2. DCUI Setup: Configure management network (IP, subnet, gateway, DNS) via DCUI
  3. Enable SSH: Activate SSH and ESXi Shell for remote management
  4. NTP Config: Set NTP servers for time synchronization
  5. vCenter Add: Add host to vCenter, join datacenter and cluster

vCenter Server Appliance Deployment

VMware · Management
  1. Stage 1 Setup: Mount vCSA ISO, run installer UI, select Install deployment type
  2. Target Selection: Connect to ESXi host, specify VM name and credentials
  3. SSO Configuration: Create SSO domain (vsphere.local) or join existing
  4. Network Settings: Assign static IP, FQDN, DNS servers, gateway
  5. Stage 2 Complete: Configure time sync, SSH access, CEIP, and verify services

HA/DRS Cluster Setup

VMware · High Availability
  1. Create Cluster: Right-click datacenter → New Cluster, name and enable HA/DRS
  2. HA Settings: Set admission control policy (% reserved capacity or failover hosts)
  3. DRS Automation: Configure automation level (Manual/Partial/Fully Automated)
  4. Add Hosts: Drag ESXi hosts to cluster, verify network and shared storage
  5. VM Overrides: Set per-VM HA/DRS rules for critical workloads

Distributed Switch Configuration

VMware · Networking
  1. Create vDS: Actions → New Distributed Switch, select version and uplink count
  2. Add Hosts: Right-click vDS → Add/Manage Hosts, select physical adapters
  3. Port Groups: Create distributed port groups for VLANs (Mgmt, VM Network, vMotion)
  4. Uplink Policy: Configure load balancing (Route based on IP hash/originating port)
  5. Security: Enable promiscuous mode if needed, set MAC address changes policy

Windows Server Hardening

Windows · Security
  1. Disable Services: Turn off unnecessary services (Print Spooler, Remote Registry)
  2. GPO Policies: Apply CIS benchmarks via GPO (password policies, account lockout)
  3. Firewall Rules: Enable Windows Firewall, restrict inbound to required ports only
  4. Patch Management: Configure WSUS/SCCM, auto-install critical updates
  5. Audit Logging: Enable advanced audit policies, forward logs to SIEM

vSphere Lifecycle Manager (vLCM)

VMware · Patching
  1. Enable vLCM: Select cluster → Updates → Lifecycle Manager → Enable
  2. Import Image: Upload ESXi ISO to vLCM depot or use online depot
  3. Create Baseline: Define desired image with patches, drivers, firmware
  4. Remediate: Check compliance, schedule remediation during maintenance window
  5. Validate: Verify all hosts compliant, check hardware compatibility issues

ESXi Patching Procedure

VMware · Maintenance
  1. Pre-Patching: Take VM snapshots, verify backups, check HCL compatibility
  2. Maintenance Mode: Put host in maintenance mode, evacuate VMs via vMotion/DRS
  3. Apply Patch: Use esxcli software vib install or vLCM remediation
  4. Reboot & Verify: Reboot host, check SSH/services, validate datastore connectivity
  5. Exit Maintenance: Exit maintenance mode, remove snapshots, monitor alerts

P2V & V2V Migration

VMware · Migration
  1. Assessment: Check disk space, network bandwidth, application dependencies
  2. VMware Converter: Install agent/agentless, configure source and destination
  3. Migration: Start conversion, monitor progress, handle VSS snapshots
  4. Post-Migration: Install VMware Tools, reconfigure network, update DNS/DHCP
  5. Validation: Test application functionality, decommission old server

Azure Arc Onboarding

Azure · Hybrid Cloud
  1. Prerequisites: Create Azure subscription, resource group, service principal
  2. Install Agent: Deploy Connected Machine agent via script/Group Policy
  3. Connect Resources: Register servers to Azure, verify portal connectivity
  4. Apply Policies: Enable Azure Policy, configure Update Management, deploy extensions
  5. Monitor: Set up Azure Monitor, configure alerts, review compliance status

VMware NSX Micro-Segmentation

VMware · Security
  1. NSX Preparation: Deploy NSX Manager, configure transport zones and edge nodes
  2. Logical Segments: Create overlay segments for application tiers (web/app/db)
  3. DFW Rules: Configure Distributed Firewall with micro-segmentation policies
  4. Security Groups: Create dynamic groups based on VM tags, OS type, or apps
  5. Monitor Traffic: Use NSX Intelligence/Flow Monitoring for traffic visualization

VMware Horizon VDI Setup

VMware · VDI
  1. Infrastructure: Deploy Connection Servers, configure load balancing
  2. Golden Image: Create and optimize Windows master image, install Horizon Agent
  3. Desktop Pools: Configure instant/linked clone pools with replica settings
  4. App Volumes: Deploy App Volumes Manager, create AppStacks for app delivery
  5. User Access: Configure entitlements, set up Unified Access Gateway for remote access
